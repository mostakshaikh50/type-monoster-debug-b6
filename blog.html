Session storage almost similar to locacal storage. The Main difference is local storage data doesn't expired but session storage data will be expired if browser is closed. The storage capacity of local storage is 5MB/10MB. On the other hand, The storage capacity of session storage is 5MB.

A variable exists inside or outside a block. If a variable is declared outside all functions or curly braces ({}), it exists in the global scope. The global variables can be accessed by any line of code in the program, including inside blocks. The variables that are declaredwithin the block scope are comparable to local ones. They are available within the block that they are defined.

The event loop is a mechanism that allows JavaScript to perform non-blocking operations. When an asynchronous task is started, the event loop will start running. Once the task is completed, the event loop will again check for any other tasks that need to be performed. This process will continue until all tasks have been completed.

A variable that has not been assigneda value is of type undefined. A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value. A function returns undefined if a value was not returned.
